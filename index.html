Short Side-Quests

The current status of these side-quests are intended for anyone, but mainly target at web-development skills. Therefore they may 'fit' best for students with CP1 experience or students who are new to computer science.

  1. Vscode.dev and files on your Chromebook [video guide] (alt: installed VS code, LiveServer extension, and native files)
  2. Test and improve you typing speed, at W3Schools.com or another typing speed site. Target: 40 words per minute with zero errors.
  3. Draw some vector graphics in Graphite.rs or some other software.
  4. HTMX for very lightweight websites (serverless molti-page app from local files).
  5. What is an API? What is a RESTfol API? And how to use some free ones: ( National Weather Service, time, and more) [javascript 'fetch' needed]

  6. Advanced HTML and CSS

    1. CSS challenges: complete a minimum of 8 CSS challenges, start with the tutorial.
    2. Design your own Responsive web page, using CSS Flexbox as the core layout tool (along with basic box-model properties). There are lots of tutorials and explainers to help.
    3. Design your own Responsive web page, using CSS Grid as the core layout tool (along with basic box-model properties). There are lots of tutorials and explainers to help.
    4. Embed a scalable vector graphic [.svg] on a webpage:
      • As an image (from a nearby SVG file) ‚Üê very basic
      • Copy/paste the SVG code into the HTML
      • Change the appearance (usually colors) of the SVG with CSS
